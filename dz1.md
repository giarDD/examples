# Заголовки

Чтобы сделать заголовок нужно ввести хэштег и писать дальше

Пример: # Вот так

Чтобы сделать заголовок меньше нужно ввести 2 хэштега

Пример: ## Вот так


# Оформление текста

Чтобы писать курсивом выделяй текст звездочкой

Пример: *Текст*

Чтобы писать жирно выделяй двумя 

Пример: **Текст**
